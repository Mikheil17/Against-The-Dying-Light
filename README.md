# Against The Dying Light
Survive until dawn by exploring the dark forest and finding scattered poems to keep your fire alive.

## What It Is
A short Unity game about survival and page collection.

## Tech
Unity | C#

## Context
My submission to the Game Design class's annual GameJam.
